# Movies-DB

A Next.js movie search application using the Free Movie Database API.

## Features

- Search movies by title
- Display movie posters, ratings, genres, and plots
- Featured movies on homepage
- Responsive design with Tailwind CSS

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **API**: Free Movie Database (imdb.iamidiotareyoutoo.com)

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

### Production

```bash
npm start
```

## Deployment

This project is deployed on Vercel. Push to the main branch to trigger automatic deployment.

## API

This project uses the [Free Movie Database API](https://imdb.iamidiotareyoutoo.com) - an open source movie database requiring no API key.
